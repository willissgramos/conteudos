Imagens e links

Caminho Absoluto:Um caminho absoluto é um caminho completo para o arquivo, incluindo o protocolo (http ou https), o domínio e o caminho completo para o arquivo no servidor. Por exemplo, src="https://www.seusite.com/imagens/sua-imagem.jpg"

Caminho Relativo:Por outro lado, um caminho relativo é um caminho para o arquivo em relação ao arquivo HTML atual. Por exemplo, se sua imagem estiver na mesma pasta que seu arquivo HTML, você poderia usar src="sua-imagem.jpg".

Atributo src: O atributo src (source) é usado para especificar o caminho para a imagem que você deseja incorporar.
Atributo alt: O atributo alt (alternative) é usado para fornecer um texto alternativo para a imagem, que será exibido se a imagem não puder ser carregada por algum motivo.
Atributos width e height: Os atributos width (largura) e height (altura) são usados para especificar as dimensões da imagem, se você colocar apenas a largura WIDTH a altura ira ficar proporcional
