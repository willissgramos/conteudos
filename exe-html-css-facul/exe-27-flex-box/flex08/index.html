Controle do tamanho de itens flexíveis

Propriedade: Flex-shrink (Encolher) e flex-grow (Crescer) (APENAS NOS ITENS)-
Por padrão os valores são Flex-shrink = 1 e flex-grow =0
Qualquer valor DIFERENTE de 0 e positivo significa que PODE encolher (Flex-shrink)
Todos os itens com valor 0 significa que NÃO PODE CRESCER (flex-grow)


Exemplo Flex-shrink Valor 0



Exemplo: Conteiner com menos de 450px, os elementos encolheram para poder caber isso se da pelo Flex-shrink ter valor igual a 1 pois por padrao pode encolher


Exemplo: Conteiner com mais de 450px, nao aconteceu nada com os elementos pois o flex-grow esta com valor igual a 0 por padrao não pode crescer
