Aqui está a estrutura básica de um formulário HTML:

A tag FORM possui dois atributos muito importantes:

action: Especifica a URL para onde os dados do formulário devem ser enviados após o envio.
method: Define o método HTTP usado para enviar os dados. Os valores comuns são “get” e “post”.

Exemplo Prático:Formulário de Login

Para entender melhor, vamos criar um formulário de login. Este formulário terá campos para o nome de usuário e a senha e um botão para enviar os dados.






Neste exemplo: Usamos a tag label para adicionar rótulos aos campos de entrada. O atributo for na tag label deve corresponder ao atributo id do elemento de entrada relacionado.
A tag input é usada para criar elementos de entrada. O atributo type define o tipo de entrada e o atributo name define o nome do elemento de entrada. Dentro da tag input, usamos diferentes tipos para diferentes campos:
type="text": Define um campo de texto de linha única.
type="password": Define um campo de entrada para a senha, onde o texto é automaticamente mascarado.
type="submit": Define um botão de envio. Quando o usuário clica neste botão, o formulário é enviado para a URL definida no atributo action do formulário.

Exercício: Crie um formulário para solicitar feedback de usuários. Ele deve conter campos para nome, endereço de e-mail e um campo de texto para o feedback. Inclua também um botão de envio.
