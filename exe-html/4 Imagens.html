O atributo src especifica o caminho ou URL da imagem. Pode ser um caminho relativo ou absoluto.
O atributo alt fornece uma descrição alternativa da imagem. É exibido caso a imagem não possa ser carregada ou para usuários com deficiência visual.
Caminhos Relativos e Absolutos:
Caminho Relativo: O caminho relativo refere-se à localização da imagem em relação à página HTML atual. Por exemplo, se a imagem estiver na mesma pasta que a página HTML, podemos usar apenas o nome do arquivo. Se estiver em uma pasta diferente, devemos incluir o caminho relativo até a imagem. Caminho Absoluto: O caminho absoluto refere-se à localização completa da imagem, incluindo o protocolo (http:// ou https://) e o domínio. É usado quando a imagem está em um site externo. Parte 2: Atributos de Imagem


Atributo width e height: O atributo width define a largura da imagem em pixels. Por exemplo: width="300". O atributo height define a altura da imagem em pixels. Por exemplo: height="200". É importante especificar o tamanho da imagem para evitar o redimensionamento indesejado e garantir que a página seja carregada corretamente. Atributo alt (Texto Alternativo): O atributo alt fornece uma descrição alternativa da imagem. Boas Práticas ao Usar Imagens em HTML:

Otimize o tamanho da imagem para garantir um carregamento rápido da página. Use formatos de imagem adequados. Por exemplo, JPEG para fotografias e PNG para gráficos com fundo transparente. Forneça textos alternativos descritivos para todas as imagens. Utilize atributos width e height para evitar que a página seja redesenhada enquanto as imagens são carregadas. Evite o uso excessivo de imagens para não sobrecarregar a página. Utilize o atributo target="_blank" ao vincular a imagens externas para abrir o link em uma nova janela, evitando que os usuários saiam completamente do seu site.
